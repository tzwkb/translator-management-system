## Overview

 Translator lifecycle and settlement management system with a FastAPI + SQLite backend, single-file HTML frontend, and companion automation agent skill.

## Key Capabilities

- Manages translator records, projects, rates, and settlement.
- Provides a lightweight backend and single-file frontend.
- Includes an automation skill for common operations.
